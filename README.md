# User Management System
# #### Guide to run the application

```
composer install
```

#### Copy .env file

```
copy .env.example .env
```

#### Set Database Detail

```
php artisan migrate
```

#### Install NPM Dependencies

```
npm install
```

#### Start Laravel Project

```
php artisan serve
```

```
npm run dev
```
#### Test Laravel Project

```
php artisan test
```
